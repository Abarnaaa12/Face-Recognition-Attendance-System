# Face Recognition Attendance System

## Project Overview
A real-time attendance management system developed using Python, OpenCV, and FaceNet.

## Features
- Face Detection
- Face Recognition
- Automatic Attendance Logging
- Duplicate Prevention
- Real-Time Processing

## Technologies Used
- Python
- OpenCV
- FaceNet
- NumPy
- SQLite/MySQL

## Applications
- Educational Institutions
- Offices
- Training Centers

## Future Scope
- Raspberry Pi Deployment
- Cloud Database Integration
- Mobile App Support
